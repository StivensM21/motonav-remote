# Прошивка MotoNav — как залить на плату

Плата: **Waveshare ESP32-S3-Touch-LCD-1.28**

## Подготовка (один раз)

1. Установи **Arduino IDE 2.x** — https://www.arduino.cc/en/software
2. **Файл → Настройки → Дополнительные ссылки для Менеджера плат**, вставь:
   ```
   https://espressif.github.io/arduino-esp32/package_esp32_index.json
   ```
3. **Инструменты → Плата → Менеджер плат** → найди `esp32` → установи
   «esp32 by Espressif Systems».
4. **Скетч → Подключить библиотеку → Управлять библиотеками** → найди
   `GFX Library for Arduino` (автор moononournation) → установи.
5. Там же найди `TinyGPSPlus` (автор Mikal Hart) → установи. Нужна для
   автономного режима (чтение GPS-модуля).

## GPS-модуль (автономный режим)

Чтобы устройство вело маршрут без телефона, нужен GPS-модуль по UART
(u-blox NEO-6M/7M/8M и совместимые, вывод NMEA, 9600 бод).

Подключение (пины заданы в начале скетча, ПРОВЕРЬ под свою плату):

| Модуль GPS | Плата | Примечание |
|---|---|---|
| TX  | GPIO 44 (`GPS_RX_PIN`) | данные из GPS в плату |
| RX  | GPIO 43 (`GPS_TX_PIN`) | обычно можно не подключать |
| VCC | 3V3 (или 5V, смотри модуль) | питание |
| GND | GND | земля |

`GPS_RX_PIN`/`GPS_TX_PIN` в начале скетча меняй под реально свободные
пины своей платы. Первый холодный старт GPS на улице — до нескольких минут.

### Как пользоваться автономным режимом

1. На странице построй маршрут («Построить маршрут и поехали»).
2. Нажми «Загрузить в устройство (автономно)» — маршрут запишется в память.
3. Убери/выключи телефон. Устройство поймает свой GPS (на экране
   «searching GPS…»), затем поведёт по маршруту сам.
4. Слева на экране навигации — зелёная точка: признак автономного режима.
5. Съехал с маршрута — покажет «OFF ROUTE». Перестроить маршрут офлайн
   нельзя (нет карт) — вернись на линию или переподключи телефон.

## Настройки платы (Инструменты)

| Параметр | Значение |
|---|---|
| Board | **ESP32S3 Dev Module** |
| USB CDC On Boot | **Enabled** |
| Flash Size | **16MB (128Mb)** |
| Partition Scheme | **Huge APP (3MB No OTA/1MB SPIFFS)** |
| PSRAM | **QSPI PSRAM** |

## Заливка

1. Открой `MotoNavDevice.ino` в Arduino IDE.
2. Подключи плату по USB-C, выбери появившийся порт (Инструменты → Порт).
3. Нажми **Загрузка** (стрелка →).
4. Если порт не появился или заливка не идёт: зажми кнопку **BOOT**,
   коротко нажми **RESET**, отпусти BOOT — и снова нажми Загрузка.
   После заливки нажми RESET.

## Батарея

Разъём батареи на плате — MX1.25. **Проверь полярность по маркировке
у гнезда перед подключением** — перепутанные плюс и минус сжигают плату.
Заряжается сама, когда плата подключена к USB.

Если процент батареи на странице заметно врёт — подстрой `BAT_CAL`
в начале скетча (0.95…1.05) и залей заново.

## Что делает прошивка

- Показывает манёвр (стрелка), дистанцию и скорость с веб-страницы.
- Отдаёт странице уровень батареи и вольтаж.
- Настройки со страницы: спящий режим (от 2 минут / никогда) и яркость —
  сохраняются и переживают выключение.
- Без связи с телефоном засыпает через заданное время; будится касанием экрана.
